# Client Communication Skill

## Description
Manages all client communications including emails, calls, meetings, and follow-ups. Helps maintain professional relationships and ensures timely responses.

## Capabilities
- Log all client communications
- Track communication history
- Set follow-up reminders
- Draft professional emails and messages
- Prepare meeting agendas
- Document meeting notes and action items
- Manage communication preferences
- Track response times

## Example Usage

### Log a Client Call
```
Log a client call with TechCorp today. Discussed project timeline, agreed to weekly status updates, and confirmed budget approval. Follow up needed in 3 days to send project plan.
```

### Draft Professional Email
```
Draft an email to the client updating them on project progress. Mention that the homepage design is 80% complete and we're ahead of schedule. Ask for feedback on the latest mockups.
```

### Get Communication History
```
Show me the last 10 communications with TechCorp client, including dates and summaries.
```

### Prepare Meeting Agenda
```
Create a meeting agenda for tomorrow's client kickoff meeting. Include introductions, project overview, timeline review, and Q&A.
```

## Communication Templates

### Project Update Email
Subject: [Project Name] - Weekly Progress Update

Body:
- Summary of work completed
- Current status and milestones achieved
- Next steps and upcoming deliverables
- Any blockers or concerns
- Questions or feedback needed

### Status Meeting Agenda
1. Review previous action items
2. Current progress update
3. Upcoming milestones
4. Discussion items
5. Next steps and assignments

## Integration
Integrates with Client MCP Server to maintain communication logs and client relationship history.

## Best Practices
- Log all significant communications
- Respond to client inquiries within 24 hours
- Send weekly progress updates
- Keep communication professional and clear
- Document all decisions and agreements
- Set follow-up reminders
- Use templates for consistency
