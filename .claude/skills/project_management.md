# Project Management Skill

## Description
Manages freelance projects, tasks, milestones, and deliverables. Helps track project progress, estimate timelines, and manage workload.

## Capabilities
- Create and manage projects with timelines and budgets
- Break down projects into tasks and subtasks
- Track task status and completion
- Log time spent on tasks
- Generate project status reports
- Calculate project completion percentages
- Monitor budget vs. actual hours
- Identify blockers and risks

## Example Usage

### Create a New Project
```
Create a new project called "Website Redesign" for client TechCorp with a budget of $10,000 and 100 hour estimate.
```

### Add Tasks to Project
```
Add these tasks to the Website Redesign project:
1. Design homepage mockup (8 hours)
2. Develop responsive layout (16 hours)
3. Implement user authentication (12 hours)
4. Create admin dashboard (20 hours)
5. Testing and bug fixes (10 hours)
```

### Log Time
```
Log 4 hours today on the "Design homepage mockup" task for the Website Redesign project.
```

### Get Project Status
```
What's the current status of the Website Redesign project? Show completion percentage, hours used, and budget remaining.
```

## Integration
This skill integrates with the Work MCP Server to manage projects and tasks. It can also integrate with the Billing MCP Server to track billable vs. non-billable hours.

## Best Practices
- Break large projects into smaller, manageable tasks
- Update task status regularly
- Log time daily for accurate tracking
- Set realistic estimates
- Review project status weekly
- Communicate progress to clients regularly
