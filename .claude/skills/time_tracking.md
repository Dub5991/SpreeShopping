# Time Tracking Skill

## Description
Accurately tracks time spent on projects and tasks. Provides insights into productivity, profitability, and time allocation across different activities.

## Capabilities
- Log time entries with descriptions
- Track billable vs. non-billable hours
- Generate timesheet reports
- Analyze time by project, client, or task
- Calculate utilization rates
- Identify time-consuming activities
- Set time estimates and compare to actuals
- Export timesheets for invoicing

## Example Usage

### Log Time Entry
```
Log 2.5 hours today working on "Design homepage mockup" for the Website Redesign project. This is billable time.
```

### Get Weekly Summary
```
Show me my time summary for this week. Break it down by project and show billable vs. non-billable hours.
```

### Analyze Project Time
```
How much time have I spent on the Website Redesign project compared to the original estimate? Am I on track or over budget?
```

### Generate Timesheet
```
Create a timesheet for last week showing all time entries, organized by project and client.
```

## Time Entry Format

### Basic Entry
- Date: 2024-01-15
- Project: Website Redesign
- Task: Homepage design
- Duration: 2.5 hours
- Billable: Yes
- Description: Created initial homepage mockup with hero section and navigation

### Detailed Entry
```
Date: January 15, 2024
Client: TechCorp
Project: Website Redesign
Task: Frontend Development
Time: 09:00 AM - 12:30 PM (3.5 hours)
Billable: Yes
Rate: $125/hour
Description: Implemented responsive navigation bar with mobile menu, added accessibility features, and integrated with design system.
```

## Time Tracking Methods

### Real-Time Tracking
Start a timer when beginning work, stop when finished. Most accurate but requires discipline.

### Retrospective Logging
Log time at end of day or week. Less accurate but more flexible.

### Time Blocking
Schedule time blocks for tasks, then confirm actual time spent.

## Time Categories

### Billable Time
- Client project work
- Client meetings
- Deliverable creation
- Revisions and iterations

### Non-Billable Time
- Administrative tasks
- Proposal writing
- Learning and development
- Marketing and sales
- Internal meetings
- Email and communication

### Investment Time
- Portfolio development
- Skill building
- Tool setup and optimization
- Process improvement

## Key Metrics

### Utilization Rate
```
Utilization = Billable Hours / Total Work Hours × 100%
Target: 70-80% for freelancers
```

### Effective Hourly Rate
```
Effective Rate = Total Revenue / Total Hours Worked
Compare to nominal rate to understand profitability
```

### Time to Revenue
Track time between work performed and payment received.

## Best Practices
- Log time daily, not weekly
- Be specific in descriptions
- Round to realistic increments (0.25 hours)
- Distinguish billable vs. non-billable
- Review time logs before invoicing
- Compare estimates to actuals
- Track all work time, not just billable
- Use categories consistently
- Set realistic time budgets
- Analyze trends monthly

## Common Time Tracking Mistakes
- Not tracking non-billable time
- Waiting too long to log time
- Being too vague in descriptions
- Not distinguishing between tasks
- Forgetting to track small tasks
- Rounding incorrectly
- Not reviewing time before invoicing
